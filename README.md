# 📱 Power BI Mobile Sales Dashboard

An interactive Mobile Sales Analytics Dashboard built using Power BI to analyze mobile sales performance across different cities, brands, payment methods, customer ratings, and time periods.

---

## 🚀 Project Overview

This dashboard provides business insights into mobile sales using interactive visualizations, KPI cards, slicers, and DAX measures. It helps analyze sales trends, customer behavior, and product performance.

---

## 🛠️ Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Power Query Transformations

- Cleaned and transformed raw data
- Changed data types
- Removed unnecessary columns
- Handled null values
- Renamed columns
- Created date-related columns
- Prepared data for analysis

---

## 📈 DAX Measures Used

- Total Sales
- Total Quantity
- Total Transactions
- Average Sales

---

## 📌 Dashboard Features

- KPI Cards
- Sales by City (Map)
- Sales by Month (Line Chart)
- Sales by Payment Method (Donut Chart)
- Customer Ratings Analysis
- Top Mobile Models
- Sales by Day Name
- Brand Filter
- Mobile Model Filter
- Payment Method Filter
- Month Slicer
- Interactive Dashboard
